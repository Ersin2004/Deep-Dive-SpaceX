# SpaceX booking website
The SpaceX Booking Website is a web application designed to facilitate ticket sales and passenger management for SpaceX spaceflights. It provides an intuitive platform for customers to explore available space destinations, book tickets, and manage their flight details.

## Requirements
Before you install the project make sure you meet the following rewuirements
- PHP 8 or higher
- Node.js
- Xampp


## Installation
- Clone the project
- Obtain the SQL code for creating the database and tables. You can find the SQL code in the provided database.sql file.
- Open phpMyAdmin or any MySQL database management tool. Create a new database named spacex (or any desired name). Run the SQL code in the newly created database to create the necessary tables and set up the database structure. Configure the database connection settings in the application to match your database credentials.
- Cd into the project directory
- Install npm dependencies
```bash
npm install
```

- Start development server
```bash
npm run dev
```

# SpaceX

Wij samen gaan samen een applicatie ontwikkelen waarop je een ticket kan booken voor een vlucht met de SpaceX. Om een vlucht te kunnen booken moet je wel een gezondheids formulier hebben waarop staat dat je gezond bent en geen of ander problemen hebt wat kan vervolgen door de vlucht.

# Onze applicatie gaat er als volgt uit zien:

- Begin pagina met een introductie over de SpaceX.
- registratie pagina om een account aan te kunnen maken.
- Een login pagina (Wachtwoord vergeten).
- Pagina met herrinering van de gezondheidsverklaring. Zo niet staat er een link waar je de gezondheidsverklaring aan kan vragen.
- Pagina waar je de vlucht kan booken.
- message van succesvolle booking.

# Advantages

- Applicatie werkt volgens de wet (gezondheidsverklaring).
- Applicatie is overzichtelijk/duidelijk.
- Applicatie geeft aan als je booking gelukt is.

# Disadvantages

- Op de applicatie staat er helaas hoe erg de uitstoot is van de vlucht.
- Applicatie kan jou niet garanderen dat alles goed gaat tijdens de vlucht.
- Applicatie is nog uiterst nieuw en kan niet zijn zoals iedereen gewild heeft. Feedback staa we open voor.

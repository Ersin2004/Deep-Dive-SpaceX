## BRAINSTORM IDEAS FOR THE SPACEX WEBSITE.

# IDEAS
landing page : 
- een landing page om een korte uitleg te geven over het space tavel programma met insights over co2 uitstoot en hoe de reis in zijn werking gaat
- de voordelen van de landing page zijn dat de klant vooraf zichzelf kan inlezen en zo bekend kan worden met het idee van ruimtereizen. Ook trekt het mogelijk nieuwe klanten
- een nadeel hiervan is er niet echt

Login page : 
- een login page zodat klanten die een geldig id hebben kunnen inloggen om zo hun reizen te boeken.
- een voordeel hiervan is dat wij als "bedrijf" zo kunnen bijhouden wie een geldig id heeft om ook daadwerkelijk een reis in te plannen.

Booking page : 
- een booking page waarop de klanten de vluchten kunnen boeken en plannen
- het voordeel hiervan is dat de klanten precies kunnen zien op welke datum/tijd er vluchten beschikbaar zijn.

FAQ pagina : 
- een pagina waar alle veelgestelde vragen op staan met antwoorden erbij, verder kunnen klanten hier ook nieuwe vragen posten en om hulp vragen.

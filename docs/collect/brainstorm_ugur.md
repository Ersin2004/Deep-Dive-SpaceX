# SpaceX


### Inleiding

We gaan een landingspagina voor SpaceX maken waarmee gebruikers gemakkelijk een vlucht naar ergens anders kunnen boeken. De landingspagina verwijst je door naar een kalendersysteem die beschikbare vluchten weergeeft en gebruikers hun gewenste dag en bestemming laat kiezen. Het bedoeling is dat de gebruiker ook de prijs van de vlucht kan zien.


### Advantages

- Het project voldoet aan het wet (gezondheidsverklaring).
- Het kalender zorgt ervoor dat het duidelijk en overzichtelijk is.
- Het project is makkelijk te gebruiken.


### Disadvantages

- Het project is niet makkelijk te maken.
- Het project kan veel tijd kosten.
- Het project kan veel geld kosten.
- Het project kan veel onderhoud nodig hebben.
